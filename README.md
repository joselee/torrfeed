### TORRfeed
Torrent aggregation app. 

#### Prerequisites
* node
* npm
* mongodb

#### Setup and run in development mode
* open up three shells
* first shell:
  * `mongod`
* second shell:
  * `cd backend`
  * `npm install`
  * `npm start`
* third shell:
  * `cd front`
  * `npm install`
  * `npm start`
